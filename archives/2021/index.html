ubuntu20安装ESP-IDF

vincen20 发表于 2021-04-29    标签：

title:在ubuntu20.10下安装eps-idf

参考视频
https://www.bilibili.com/video/BV19U4y1b7v6
安装的esp-idf是4.2
首先安装依赖环境

sudo apt-get install git wget flex bison gperf python3 python3-pip python3-setuptools cmake ninja-build ccache libffi-dev libssl-dev dfu-util libusb-1.0-0

设置串口权限

sudo usermod -a -G dialout $USER

下载idf

git clone -b v4.2 --recursive https://github.com/espressif/esp-idf.git

设置环境变量

打开文件 

   nano ~/.bashrc
   alias get_idf='.$HOME/esp/esp-idf/export.sh'
export PATH="/home/vin/esp/esp-idf/components/esptool_py/esptool:$PATH"
export PATH="/home/vin/esp/esp-idf/tools:$PATH"
export PATH="/home/vin/esp/xtensa-esp32-elf/bin:$PATH"

保存，退出
现在就可以测试了

  idf.py menuconfig
idf.py build
idf.py flash
idf.py monitor
